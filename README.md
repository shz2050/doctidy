# Doctidy

A document tidying skill that brings order to chaos. Designed for both human users and AI agents to maintain a sustainable document management system.

## Features

- **Scoring System**: Evaluate document organization quality with detailed metrics
- **Entropy Control**: Prevent document chaos with automated rules
- **Auto-categorization**: Automatically classify and organize new files
- **Agent-Ready**: Designed for AI agent readability and usage
- **Path-Agnostic**: No hardcoded paths - works with any target directory

## Commands

| Command | Description |
|---------|-------------|
| `init` | Initialize doctidy infrastructure |
| `score` | Evaluate and score the document repository |
| `analyze` | Deep analysis of organization issues |
| `organize` | Execute cleanup based on analysis |
| `intake` | Auto-categorize new files |
| `status` | Quick status overview |
| `reset` | Remove all doctidy infrastructure |

## Quick Start

```
# Initialize in your target directory
/doctidy init

# Score current organization
/doctidy score

# Analyze issues
/doctidy analyze
```
